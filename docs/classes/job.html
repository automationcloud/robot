Job

Unified interface for running Autopilot scripts.

This class has two standard implementations:

  • LocalJob runs the job using local Autopilot Engine and Chromium browser and tracks its progress by subscribing to script lifecycle events.
  • CloudJob submits the job to Automation Cloud and tracks its progress Automation Cloud API.

The job should be created via Robot.createJob.
